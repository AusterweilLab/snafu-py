- These functions extend phonemic/letter fluency analysis to identify homonyms an rhymes (so as to include them as part of the same cluster)

- These functions have not been incorporated into SNAFU yet

- It relies on the CMU Pronunciation dictionary, which has been augmented

- The code was written by David Clark (clarkdg@iu.edu) and supplied by Justin Bushnell (jbushnel@iu.edu)

- Justin: "For computing rhymes, a level needs to be set to determine how strict the rhyme must be in order to be connected. It is currently set at 2 (on a 1-5 scale with 5 being the most strict). We found that level 2 incorporates the most sensible slant rhymes."
